Versions
--------

Datasets have a collection of versions, points in time to which you can
roll back.

See `API Reference - Create savepoint <https://crunch.io/api/reference/#post-/datasets/-dataset_id-/savepoints/>`_.

See `API Reference - List savepoints <https://crunch.io/api/reference/#get-/datasets/-dataset_id-/savepoints/>`_.

See `API Reference - Savepoint details <https://crunch.io/api/reference/#get-/datasets/-dataset_id-/savepoints/-savepoint_id-/>`_.

See `API Reference - Update savepoint description <https://crunch.io/api/reference/#patch-/datasets/-dataset_id-/savepoints/-savepoint_id-/>`_.

See `API Reference - Revert dataset to savepoint <https://crunch.io/api/reference/#post-/datasets/-dataset_id-/savepoints/-savepoint_id-/revert/>`_.
